# Live depoyed app link:
https://app-city-pulse.netlify.app

## Features

View list of events

View event details

Map preview on event page

Login & Signup (mock authentication)

Logout option in Navbar when logged in

Mobile-friendly design

Clean folder structure (components, pages, hooks, context, data)


## How to Run

git clone https://github.com/mohussain237/City-Pulse.git

cd City-Pulse

npm install

## Start the project

npm start

## Mock Authentication

Login and Signup use mock data stored in context.

## No backend required.

- Map Preview

Event details page shows a static map preview using Google Maps iframe.
